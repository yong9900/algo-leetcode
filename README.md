# Algo LeetCode

This project is a Java-based playground mainly created to write, run, and test source code for algorithms and solve LeetCode problems.

## Project Structure

- **Source Code**: All main algorithm implementations are located under [src/main/java/com/leetcode](file:///Users/mikezhao/tries/github/algo-leetcode/src/main/java/com/leetcode) and [src/main/java/com/algo](file:///Users/mikezhao/tries/github/algo-leetcode/src/main/java/com/algo).
- **Unit Tests**: Unit tests are located under [src/test/java/com/leetcode](file:///Users/mikezhao/tries/github/algo-leetcode/src/test/java/com/leetcode) and [src/test/java/com/algo](file:///Users/mikezhao/tries/github/algo-leetcode/src/test/java/com/algo).

## Technologies Used

- **Java 26**: Core programming language.
- **Maven**: Build and dependency management tool.
- **JUnit 5 & AssertJ**: Testing frameworks used to verify correctness of algorithms.
- **Google Guava & Apache Commons**: Core utility libraries.
- **Project Lombok**: Reducing boilerplate code.

## Getting Started

To run the unit tests, verify dependencies, and compile the code, you can use the standard Maven commands in your terminal:

```bash
# Clean and run tests
mvn clean test
```
